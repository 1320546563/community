## 部署
### 依赖
- 安装git  
- 安装jdk  
- maven  
- mysql
## 步骤   
- yum install git 
- mkdir App
- cd App
- git clone https://github.com/1320546563/community.git  
- yum install maven  
- mvn clean compile package
